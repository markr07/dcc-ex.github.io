TrainNavigator
*******************

Simple Native DCC-EX protocol based controller for model railroads

* multiple throttles to control locos
* turnouts controller defined in DCC-EX setup
* layout editor with turnout control


The app needs only access to local network permission to connect to an EX-CommandStation.
